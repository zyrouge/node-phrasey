# Project Structure

A basic project contains mainly of the [configuration file](./configuration.md), [schema file](./schema.md) and the [translations files](./translation.md). Additionally, [hooks](./hooks.md) can be included.

```
.
├── i18n
│  ├── en.json
│  └── [...].json
├── phrasey-config.json
└── phrasey-schema.json
```
